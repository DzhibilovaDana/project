# FPGA Pin Configuration Web Interface

Веб-приложение для визуального конфигурирования FPGA плат.

## Описание

Проект представляет собой веб-интерфейс для:
- Визуального представления схемы FPGA
- Переназначения пинов на схеме
- Генерации Verilog файлов
- Прошивки платы через Quartus

## Структура проекта

- `frontend/` - React приложение
- `backend/` - Python бэкенд
- `docs/` - документация

## Технологии

- Frontend: React
- Backend: Python
- FPGA Tools: Quartus
- Version Control: Git
